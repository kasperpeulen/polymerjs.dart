#polymerjs_examples

Instructions:

* Make sure your terminal is in the polymerjs_examples directory.
* Run `pub get`.
* Run `bower install`.
* Run `pub serve`.
* Then go to [http://localhost:8080](http://localhost:8080)

Or if you are familiar with grinder:

* `grind examples`
* Then go to [http://localhost:8080](http://localhost:8080)
